# Clue Images

Drop your clue pictures here as:

- `clue1.jpg`
- `clue2.jpg`
- ...
- `clue10.jpg`
- `treasure.jpg` ‚Äî shown on the "You Found the Treasure" page

The game references them in `game.js` (see the `CLUES` array). Any
common image format works ‚Äî just update the file extension in `game.js`
if you use something other than `.jpg`.

While images are missing the page will show a dashed orange placeholder box.
